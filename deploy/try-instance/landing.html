Try Syntra

This is a shared demo instance of Syntra running the four flagship capsules — predictive autoscaling, anomaly-aware routing, seasonal fraud thresholds, and hierarchical region routing. Anyone with the link can issue /decide and /feedback calls; there is no auth.

Your data is wiped daily at 00:00 UTC. The capsules' learned weights, decision logs, and any state you push reset to zero on the dot. Don't put anything here you wouldn't post on a public bulletin board.

For production use, run your own instance — the image is one docker run away. See the deployment guide for the full walkthrough.
